Kooperative Code-Review-Plattform
Projektbeschreibung
Dieses Projekt ist eine kooperative Code-Review-Plattform, die die Zusammenarbeit von Programmierern fördert, indem sie Echtzeit-Codebearbeitung, Syntax-Hervorhebung und einen integrierten Chat bereitstellt. Der Fokus liegt auf der manuellen Analyse und Verbesserung von Algorithmen, ohne dass automatische Tools die Arbeit übernehmen.
Die Plattform wurde mit HTML, CSS, JavaScript, Node.js, Express und Socket.IO umgesetzt und ermöglicht es mehreren Nutzern, gleichzeitig an Code zu arbeiten und direkt Feedback zu geben. Ziel des Projekts ist es, die Kommunikation, Zusammenarbeit und die Fähigkeit zur selbstständigen Verbesserung von Code zu fördern.
Funktionen
Echtzeit-Codebearbeitung zwischen mehreren Nutzern
Syntax-Highlighting mittels Prism.js
Chatfunktion für Kommunikation und Feedback
Unterstützung verschiedener Programmiersprachen im Editor
Keine automatischen Tools – der Mensch steht im Mittelpunkt
Motivation
Das Projekt entstand im Rahmen einer wissenschaftlichen Arbeit zum Thema kooperative Systeme. Es zeigt, wie digitale Werkzeuge die kollaborative Arbeit an Algorithmen erleichtern können, ohne die analytische Leistung des Menschen zu ersetzen.
Technologien
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
Echtzeitkommunikation: Socket.IO
Syntax-Highlighting: Prism.js

Um den Code abzurufen, erst die nötigen Packages installieren -> dann über node den Server laufen lassen -> Dann über den localport im Browser die Website öffnen.
